# Overview

LightRoutine is a routine tracking mobile app for iOS and Android.

## Goal

Enable users to create daily routines and track their completion over time.
Provide statistics and streak tracking to motivate consistent habit building.

## Tech Stack

- Mobile: Expo (React Native) + Expo Router
- Backend: NestJS + Prisma + PostgreSQL
- State: Zustand (client) + TanStack Query (server)
- Deploy: Railway (backend) + EAS Build (mobile)
- Monorepo: npm workspaces

## Status

MVP in development.
