# Requirements

## MVP Features

### Authentication
- Email registration and login
- JWT-based authentication (access + refresh tokens)
- Logout

### Routine Management
- Create routines with name, category, time slot, color, repeat days
- Edit and soft-delete routines
- View routines grouped by time slot
- Reorder routines

### Daily Tracking
- Toggle routine completion per day
- View completion status by date

### Statistics
- Today completion rate
- Weekly completion rate with daily breakdown
- Monthly calendar view (grass-style heatmap)
- Streak tracking (current and longest)

### Screens
- Home: today's routines with check toggles
- Calendar: monthly view with date selection
- Statistics: charts and streak display
- Settings: profile info and logout

## Post-MVP (not in scope)
- Social login
- Push notifications
- Routine templates
- Numeric/timer tracking
- Dark mode
