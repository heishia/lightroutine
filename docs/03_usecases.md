# Use Cases

## Auth
1. Register with email, password, nickname
2. Login with email, password -> receive tokens
3. Refresh expired access token
4. Logout (client discards tokens)

## Routines
1. Create a new routine with attributes
2. View all active routines grouped by time slot
3. Edit routine attributes
4. Delete routine (soft delete, preserves history)
5. Reorder routines via drag-and-drop

## Tracking
1. Toggle today's routine completion
2. View completion status for any date

## Statistics
1. View today's completion rate
2. View weekly completion rates with bar chart
3. View monthly calendar heatmap
4. View current streak and longest streak
